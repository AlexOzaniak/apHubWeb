Jednoduché a čisté

A-P-SmartHub je projekt pre zjednotené riadenie inteligentnej domácnosti postavený na C# a .NET Framework.

Zobraziť na GitHub-e Pozrieť galériu
Hlavný telemetrický dashboard (Main Dashboard) Prehľad teplôt, stavu plynu, vlhkosti, chýb a pripojených zariadení.

O projekte

A-P-SmartHub funguje ako centrálna brána spájajúca hardware a software.

Čo robí

Prepája hardvér a softvér, poskytuje dôležité informácie o domácnosti, ako sú plyn, teplota a vlhkosť. Dokáže ovládať svetlá a taktiež zobrazuje, čo vám hrá na Spotify.

Komu pomáha

každému, kto chce mať jednoduché a čisté riešenie pre riadenie svojej inteligentnej domácnosti.

Čo využíva

Objektovo orientované natívne štruktúry v C#, runtime pipeliny .NET a asynchrónne slučky vykonávania kódu.

Rozhranie aplikácie & Dokumentácia

Kliknutím na ktorúkoľvek kartu nižšie otvoríte responzívny celoobrazovkový prehliadač.

AI Chat asistent Integrovaná vrstva umelej inteligencie pre ovládanie domova.
Main Dashboard Centrálny monitorovací panel celého prostredia systému SmartHub.
All Devices Kompletná hardvérová správa, prehľad a manažment pripojených zariadení.
Fyzické ESP32 hardvérové riešenie Reálna laboratórna implementácia riadiaceho mikrokontroléra pre zber dát.
Climate Control Dedikované rozhranie a riadiace widgety pre nastavenie ventilácie a teploty.
Registrácia nového účtu Bezpečný autentifikačný formulár pre registráciu užívateľa.
Reset Password Mail Užívateľské a systémové rozhranie doručeného overovacieho e-mailu pre zmenu hesla.
Gas Alert Notifikácia Automatizovaný kritický e-mail pri detekcii nebezpečných environmentálnych limitov.
Užívateľský Login Bezpečná vstupná brána pre prihlásenie do aplikácie.
Light Control Interaktívne prispôsobenie osvetlenia s možnosťou dynamického nastavenia farby RGB LEDiek.
Spotify Connector Integrovaná funkcia pre prepojenie s službou Spotify a streamovanie hudby.
Settings systému Konfiguračné centrum pre správu svojho účtu a nastavení aplikácie.
